The most important callout from our Lead Linguist is about the proper usage of pre-filled Machine 
Translations. STT Full MTPE projects have high quality standards in accuracy and fluency. The original 
MT inputs must be carefully validated and edited so that the native language speaking customers cannot 
tell it’s a translation.  
 
Please find the detailed comment below:
 
87198838 filename: EnGb-XX_TDT_source_09212023.165.csv
 
1.	Mistranslations: e.g. “loved one” translated as “amante/lover”. This word has a 
romantic/sexual connotation. Please don’t trust the machine translation output and 
always choose the most adequate translation depending on the context.
    
 
2.	Avoid literal translations: “varias ocasiones de llevar” is a literal translation which 
sounds quite unnatural and unidiomatic.
    
 
3.	Avoid inappropriate words: the word “pendant” was translated as “pedante” (snobbish, 
pedantic). The use of this kind of word can have a very negative impact.
    
 
4.	Make sure that the target text makes sense in terms of semantics. This sentence is 
copying the source segment structure, but we need to make sure that the target 
segment always has the best possible quality. The original translation looked like it was 
incomplete.
    
5.	Measurement format: please always follow the corresponding style guide. There we can 
find instructions to “Convert non-metric measurements (e.g. inches or feet).” 
(https://amazontranslationlearn.com/mod/book/view.php?id=758&chapterid=12728).
    
 
6.	Lack of number accord: “Hecho” was in singular but the subject was in plural. This type 
of errors are easy to avoid by having a quick look at the target text once the post-editing 
work is finished.
    
    
7.	Unintelligible: this specific sentence was barely understandable. Don’t be afraid to 
distance yourself from the source text to convey the meaning expressed in the clearest 
way possible.
    
    
8.	Trailing space added before the period: be careful to use the correct punctuation.
    
 
87199358 filename: EnGb-XX_TDT_source_09212023.185.csv
 
9.	Review the target text to avoid spelling mistakes. Be especially careful with longer 
segments, where these mistakes may be harder to spot.
    
    
 
    
 
10.	English language false friends: “casual” doesn’t have this meaning (“informal”) in 
Spanish, so a literal translation won’t work in this context (https://dle.rae.es/casual).
    
    
11.	Lack of editing: in this case, a comma seems to have been introduced inside one of the 
names by the machine translation. Please make sure to always postedit these cases in 
order to produce a clear target text.
    
 
86014866 filename: EnGb-XX_TDT_source_09212023.103.csv
 
12.	Grammar: make sure to use correct sentence and verb structures that respect Spanish 
Grammar.
